MAPPING THE APC40

The APC40 is the world’s finest control instrument for playing Ableton Live software. That said, some of you have expressed interest in turning the APC40 into a controller for other software. Here’s how to do it. The APC40 is MIDI compliant, so as long as your software is too, you’re virtually guaranteed that they can work together.
 
BASIC
Many programs have a MIDI Learn function. Just turn this on and your software will “listen” for you to push a button, move a fader, or turn a knob. Then you can assign a software function to that controller. That’s it!
 
INTERMEDIATE
This controller map below shows every controller message and key assignment for the controls on the APC40. Look up the note number or controller message on the map and assign whatever control you want to use in your software to that controller number or key. Click the image to make it larger.
